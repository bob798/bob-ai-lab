# BobAILab 仓库介绍

BobAILab 是我的公开实验室。

这里主要记录三类内容：

- 我如何用 AI 从 0 到 1 做产品
- 我如何和 AI 协作、做决策、做需求管理
- 我如何把实践沉淀成方法论、文章和可复用工具

这个仓库不是新闻流，也不是 AI 工具收集站。

它更像一个持续演化的工作台：一边做真实项目，一边复盘判断、协作方式和系统化方法。

如果你也是独立开发者，或者正在尝试把 AI 真正放进自己的工作流，这个仓库里的内容应该会对你有帮助。

## 你能在这里看到什么

- 产品方法论：怎么判断一个方向值不值得做，怎么避免高效做错
- AI协作方法：怎么提问、怎么纠偏、怎么把 AI 用成稳定协作者
- 案例复盘：真实项目里踩过的坑、做过的判断、形成的方法

## 一句话定位

> 我用 AI 做产品，持续沉淀方法与工具。
