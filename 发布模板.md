# 发布模板

这个模板用于把一篇主文拆成适合不同平台传播的发布素材。

## 一、主文发布卡

```md
# 标题

主文链接：

一句话核心判断：

适合谁看：

- 
- 
- 

这篇解决什么问题：

- 
- 

读完后希望读者记住什么：

- 
```

## 二、短内容拆解模板

每篇主文至少拆 3 条短内容。

### 短内容 1：一句话判断

```md
标题：

正文：

结尾动作：
- 看完整版文章
- 留一个问题
- 引到下一篇
```

### 短内容 2：一个关键点

```md
标题：

结构：
- 常见误解
- 我的判断
- 为什么
- 一个例子
```

### 短内容 3：一个可执行动作

```md
标题：

结构：
- 场景
- 做法
- 最小动作
- 常见错误
```

## 三、发布后复盘模板

```md
文章：

发布时间：

平台：

最容易被记住的一句话：

最有共鸣的点：

最值得继续展开的问题：

下一篇可以接什么：
```

## 四、最适合 BobAILab 的发布组合

建议固定成这套：

- 1 篇主文
- 3 条短内容
- 1 次发布后复盘

这样既能保持深度，也不会让长文写完就沉掉。
